What is Ruby on Rails?

Ruby on Rails, also known as ROR, or simply, Rails, is a web development framework that is written using the programming language, Ruby. It extends the Ruby programming language. It was designed to make web developer's lives easier, and allows you to write less code. It makes web development more enjoyable.

To get started using Rails, you have to install the software code onto your computer using your command line, and this will add to the Ruby programming language.

Essentially, Rails takes Ruby, and combines it with HTML, CSS, and JavaScript to create a web application. You use rails to build websites. Rails makes it easier to collaborate with others and maintain web applications.

Rails has two guiding principles:

  • DRY: Don't repeat yourself
    • DRY is a principle all software developers should stick to.
    • By not repeating the same code over and over again, you minimize our risk of having bugs in your code.
  • Convention Over Configuration:
    • Rails is an opinionated software. There is a "best" way to do things, and it wants you do do it that way.

Why Rails?

Ruby and Rails is as close to English as programming gets. Ruby and Rails syntax is excellent compared to other programming languages and that makes happier programers.

Creating things in Rails is fast and easy and that's why it's so popular today. You could have an idea and have it made using rails in just a few hours. The same couldn't be said for other frameworks.
